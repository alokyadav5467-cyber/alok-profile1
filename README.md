# Alok Kumar — Cinematic Portfolio

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy on Vercel (Free)

1. Push this folder to GitHub
2. Go to vercel.com → New Project
3. Import your GitHub repo
4. Click Deploy — done!

## Swap the video

Replace `public/hero.mp4` with your new video anytime.
The video auto-plays, loops, and has mute/play controls built in.

## Customize

- Colors: edit `--neon` in `src/styles/globals.css`
- Content: edit individual components in `src/components/`
- Add sections: create new component → import in `src/app/page.tsx`
